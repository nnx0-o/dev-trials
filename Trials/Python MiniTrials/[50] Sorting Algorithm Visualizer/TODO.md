# TODO: Sorting Algorithm Visualizer

Write a Python script that demonstrates and benchmarks multiple sorting algorithms.

## Requirements
- Implement the following sorting algorithms from scratch: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, and Heap Sort
- Generate a random list of 20 integers (or accept a custom list)
- For each algorithm, print step-by-step visualization as an ASCII bar chart (normalize values to max 20 chars wide) after each major operation
- Track and display comparison count and swap count for each algorithm
- After all algorithms complete, print a comparison table: algorithm name, comparisons, swaps, and relative time taken
- Use `time.perf_counter()` for timing

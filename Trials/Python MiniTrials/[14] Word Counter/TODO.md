# TODO: Word Counter

Write a Python script that counts words, characters, and lines in a string entered by the user.

## Requirements
- Ask the user to paste or type text (single or multi-line); signal end with a blank line
- Collect all lines until a blank line is entered
- Compute:
  - Total number of lines (excluding the blank terminator)
  - Total number of words (split on whitespace)
  - Total number of characters (including spaces, excluding the newline terminator)
- Print each stat on its own line: `Lines: <n>`, `Words: <n>`, `Characters: <n>`

# TODO: Tic-Tac-Toe

Write a Python script implementing a two-player Tic-Tac-Toe game in the terminal.

## Requirements
- Represent the board as a 3x3 list of lists, initialized with position numbers 1–9
- Display the board after every move with borders
- Alternate turns between Player X and Player O
- Accept input as a number 1–9 corresponding to board position
- Validate input: reject non-integers, out-of-range values, and already-taken cells
- Check for win (rows, columns, both diagonals) after each move
- Detect draws when the board is full with no winner
- Ask to play again after each game

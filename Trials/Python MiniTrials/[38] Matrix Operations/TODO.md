# TODO: Matrix Operations

Write a Python script that performs matrix operations without using NumPy.

## Requirements
- Implement functions for: matrix addition, matrix subtraction, matrix multiplication, scalar multiplication, matrix transposition, and determinant calculation (for 2×2 and 3×3)
- Allow the user to input matrix dimensions and values row by row
- Present a menu to choose an operation; prompt for a second matrix if needed
- Pretty-print matrices with aligned columns
- Validate matrix dimensions for each operation (e.g., multiplication requires compatible sizes)
- Show step-by-step output for determinant calculations

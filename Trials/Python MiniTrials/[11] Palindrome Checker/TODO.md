# TODO: Palindrome Checker

Write a Python script that checks if a given string is a palindrome.

## Requirements
- Ask the user for a word or phrase: `Enter a string: `
- Strip leading/trailing whitespace and convert to lowercase before checking
- Ignore spaces when checking (remove all spaces)
- Print `'<original>' is a palindrome.` or `'<original>' is not a palindrome.`
- Examples: `racecar`, `A man a plan a canal Panama` → palindromes

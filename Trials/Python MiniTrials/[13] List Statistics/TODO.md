# TODO: List Statistics

Write a Python script that computes basic statistics from a list of numbers entered by the user.

## Requirements
- Ask the user to enter numbers one per line; they type `done` to finish
- Parse each entry as a float; skip and warn on invalid entries
- After input ends, compute and print:
  - Count of valid numbers
  - Sum
  - Minimum
  - Maximum
  - Mean (average), rounded to 2 decimal places
- If no valid numbers were entered, print `No valid numbers entered.`

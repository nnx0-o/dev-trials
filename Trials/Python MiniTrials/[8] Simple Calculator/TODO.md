# TODO: Simple Calculator

Write a Python script that performs basic arithmetic operations.

## Requirements
- Ask the user for two numbers (allow floats)
- Ask the user to choose an operation: `+`, `-`, `*`, `/`
- Perform the selected operation
- Print the result in the format: `<a> <op> <b> = <result>`
- Handle division by zero by printing `Error: Division by zero.`
- Handle invalid operators by printing `Error: Invalid operator.`

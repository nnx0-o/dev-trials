# TODO: Multiplication Table

Write a Python script that prints a formatted multiplication table.

## Requirements
- Ask the user for a number `n`: `Enter a number: `
- Print the multiplication table for that number from 1 to 10
- Each line should follow the format: `<n> x <i> = <result>` (right-aligned with consistent spacing)
- Example for n=5: `5 x  1 =  5`, `5 x  2 = 10`, etc.

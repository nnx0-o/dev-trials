# TODO: Countdown Timer

Write a Python script that implements a countdown timer in the terminal.

## Requirements
- Ask the user to enter the number of seconds to count down
- Validate that the input is a positive integer
- Use `time.sleep(1)` between each tick
- Display the remaining time on the same line using `\r` (carriage return) in the format: `Time remaining: <MM:SS>`
- When done, print `Time's up!` on a new line

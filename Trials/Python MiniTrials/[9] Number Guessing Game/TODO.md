# TODO: Number Guessing Game

Write a Python script where the user guesses a randomly generated number.

## Requirements
- Import `random` and generate a random integer between 1 and 100 using `random.randint()`
- Repeatedly prompt the user: `Guess a number between 1 and 100: `
- After each guess, print `Too low!`, `Too high!`, or `Correct! You guessed it in <n> attempts.`
- Count the number of attempts
- End the loop when the correct number is guessed

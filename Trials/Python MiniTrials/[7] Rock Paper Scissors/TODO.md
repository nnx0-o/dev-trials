# TODO: Rock Paper Scissors

Write a Python script that lets the user play Rock Paper Scissors against the computer.

## Requirements
- Import the `random` module
- Ask the user to enter their choice: `Enter rock, paper, or scissors: `
- Randomly select the computer's choice from `['rock', 'paper', 'scissors']`
- Print both choices
- Determine and print the winner using standard RPS rules
- Handle invalid input by printing `Invalid choice.` and exiting
- Output format: `You chose: <choice>`, `Computer chose: <choice>`, then `You win!`, `Computer wins!`, or `It's a tie!`

# TODO: Mad Libs Generator

Write a Python script that generates a funny story by prompting the user for words.

## Requirements
- Define at least one story template as a string with labeled placeholders like `{noun}`, `{verb}`, `{adjective}`, `{place}`, `{name}`, etc.
- Prompt the user for each unique placeholder in the order it first appears
- Substitute the user's inputs into the template
- Print the final story with a header: `--- Your Mad Lib ---`
- Include at least 3 different templates and pick one at random using `random.choice()`

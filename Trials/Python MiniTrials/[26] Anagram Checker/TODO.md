# TODO: Anagram Checker

Write a Python script that checks if two words or phrases are anagrams of each other.

## Requirements
- Ask the user to enter two strings
- Normalize both strings: lowercase, remove all spaces and punctuation
- Compare sorted character lists to determine if they are anagrams
- Print `'<word1>' and '<word2>' are anagrams.` or `... are NOT anagrams.`
- Also show the character frequency breakdown for each string
- Allow the user to check another pair or exit

# TODO: Fibonacci Sequence

Write a Python script that generates and prints the first N numbers of the Fibonacci sequence.

## Requirements
- Ask the user: `How many Fibonacci numbers to generate? `
- Validate that the input is a positive integer; if not, print `Please enter a positive integer.` and exit
- Generate the sequence starting with 0 and 1
- Print each number separated by a comma and space on a single line
- Example for n=7: `0, 1, 1, 2, 3, 5, 8`

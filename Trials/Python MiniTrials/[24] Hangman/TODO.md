# TODO: Hangman

Write a Python script implementing the Hangman word-guessing game.

## Requirements
- Include a hardcoded list of at least 20 words
- Pick a random word at the start of each game
- Show the current state as underscores and revealed letters: `_ _ a _ _`
- Show the hangman figure progressively (ASCII art) through 6 wrong guesses
- Track and display used letters: `Used: a, b, c`
- Accept only single alphabetic characters; reject duplicates and non-letters with a message
- Win condition: all letters revealed. Loss condition: 6 wrong guesses
- Ask to play again after each game

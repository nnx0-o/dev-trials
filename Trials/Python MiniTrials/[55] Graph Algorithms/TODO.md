# TODO: Graph Algorithms

Write a Python script implementing a graph data structure with multiple algorithms.

## Requirements
- Implement a `Graph` class supporting both directed and undirected, weighted and unweighted modes
- Methods: `add_vertex`, `add_edge`, `remove_vertex`, `remove_edge`, `neighbors`
- Implement and demonstrate: BFS, DFS, Dijkstra's shortest path, detection of cycles, topological sort (for DAGs), and connected components
- Accept a graph as an adjacency list from a text file
- After loading, present a menu to run each algorithm, prompting for start/end vertices where needed
- Visualize the graph as an ASCII adjacency matrix for small graphs (≤10 nodes)

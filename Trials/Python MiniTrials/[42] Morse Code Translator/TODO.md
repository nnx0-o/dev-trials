# TODO: Morse Code Translator

Write a Python script that translates between English text and Morse code.

## Requirements
- Define a complete Morse code dictionary for A–Z, 0–9, and common punctuation (`. , ? ! / - ( )` at minimum)
- Support two modes: `encode` (text → Morse) and `decode` (Morse → text)
- Encoding: separate letters with a single space, words with ` / `
- Decoding: reverse-lookup each Morse symbol; handle unknown symbols with `?`
- Ask the user for mode and input string
- Display the result and also write it to `morse_output.txt`

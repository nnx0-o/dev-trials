# TODO: Prime Number Checker

Write a Python script that checks whether a number is prime and lists all primes up to that number.

## Requirements
- Ask the user for a positive integer greater than 1
- Implement a function `is_prime(n)` that returns `True` if `n` is prime
- Use trial division up to the square root for efficiency
- Print whether the entered number is prime or not
- Print all prime numbers from 2 up to (and including) the entered number, comma-separated

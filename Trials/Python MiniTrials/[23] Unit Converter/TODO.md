# TODO: Unit Converter

Write a Python script that converts between multiple unit types.

## Requirements
- Support three categories: Length (km, m, cm, mm, miles, feet, inches), Weight (kg, g, lb, oz), Temperature (Celsius, Fahrenheit, Kelvin)
- Present a category menu first, then a unit selection menu
- Ask the user for the value, the source unit, and the target unit
- Perform the conversion using exact formulas
- Print the result rounded to 4 decimal places: `<value> <from_unit> = <result> <to_unit>`
- Allow the user to convert again or exit after each conversion

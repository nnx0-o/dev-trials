# TODO: Typing Speed Test

Write a Python script that measures the user's typing speed and accuracy.

## Requirements
- Include at least 10 sample sentences
- Pick a random sentence and display it to the user
- Record the start time using `time.time()` before the user begins typing
- Ask the user to type the sentence exactly: `Type the text above: `
- Record the end time after they press Enter
- Calculate: elapsed time in seconds, words per minute (WPM = words / minutes), and accuracy percentage (matching characters / total characters × 100)
- Print: time taken, WPM, and accuracy

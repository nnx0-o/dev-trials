# TODO: Basic To-Do List

Write a Python script implementing a simple to-do list with persistence to a text file.

## Requirements
- Store tasks in `todos.txt`, one task per line
- On startup, load existing tasks from the file (create if it doesn't exist)
- Menu options: `1. Add task`, `2. View tasks`, `3. Mark task done (remove)`, `4. Exit`
- View: print tasks numbered starting from 1, or `No tasks.`
- Mark done: ask for task number, remove from list, save file
- Save the list back to `todos.txt` after every modification
